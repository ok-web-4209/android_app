# Android Golf Game App

This repository contains the initial data layer and flow documentation for a golf game tracking app.

## What's Included
- Data models for players, seasons, courses, and games.
- An in-memory repository with score calculations and CSV export.
- A product flow document capturing the required UX paths.

## Next Steps
- Wire the repository into Compose UI screens.
- Add persistence (Room) and Google Maps place search for course lookup.
- Export stats to CSV/XLSX files using Android storage APIs.
